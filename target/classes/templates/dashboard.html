API Endpoints Disponibles
Gestión de Entidades:
  • GET /api/v1/entidades - Listar entidades
  • POST /api/v1/entidades - Crear entidad
  • PUT /api/v1/entidades/{id} - Actualizar entidad
  • DELETE /api/v1/entidades/{id} - Eliminar entidad
Gestión de Convenios:
  • GET /api/v1/convenios - Listar convenios
  • POST /api/v1/convenios - Crear convenio
  • PUT /api/v1/convenios/{id} - Actualizar convenio
  • DELETE /api/v1/convenios/{id} - Eliminar convenio
